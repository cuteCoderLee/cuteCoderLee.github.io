Linux

基本知识

硬件 内核（kernel） 外壳（shell：用户与内核间的命令解释器） 应用程序
sh csh ksh
特点：

  • 多任务（cpu时间分片分给不同进程)
  • 多用户（同时)
  • 管道（前一个程序的输出作为后一个程序的输入）
  • shell解释型高级语言，强大
  • 安全保护机制
  • 稳定
  • 用户界面：常用命令行界面，提供图形界面
  • 网络支持：TCP/IP协议
  • 移植性好

    用户和用户组管理

    UID：/etc/passwd（7个字段 root：x:0:0:root:/root:/bin/bash)
    GID：/etc/group (root:x:0:)
    用户查询： id finger（用户属性）
    新增用户： useradd[参数][用户名]
    设置密码：passwd
    修改属性：usermod[用户名] -d -g
    新增用户组：groupadd
    删除用户组：groupdel
    修改用户组：groupmod[-g -n][用户组名]

    文件和目录管理

    绝对路径 /
    相对路径：.当前目录 ..上一级目录
    显示当前目录：pwd
    变更： cd 没有任何参数，回到当前用户的家目录
    新增：mkdir[-m 存取模式（掩码）][-p建立其所有不存在的父目录] dirname
    删除：rmdir[-p] dirname 空目录,对其父目录具有写权限
    复制：cp源文件 目标
    移动：mv
    删除：rm[-ir]文件或目录提示
    查找：find[路径][参数] -name -user
    查看文件内容：cat（直接查看不翻页） more（翻页） less head tail

    文件系统管理

    存储和组织计算机数据的方法
    传统型文件系统：无日志，先写入数据，再写入元数据；ext2
    日志型文件系统：先写入日志记录文件；ext3 ReiserFS
    索引式：文件属性与内容存放在不同区块
    非索引式：只有block；FAT（windows）

    挂载：将文件系统与目录树关联
